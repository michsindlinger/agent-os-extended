- Preview changes before applying
- Creates backups

## Activation Modes Summary

| Mode | When to Use | Frontmatter |
|------|-------------|-------------|
| **Auto-Load** | Framework/file-type specific | Include `globs:` with patterns |
| **Explicit Only** | Utility skills | No `globs`, `always_apply: false` |
| **Always Active** | Project standards | `always_apply: true` |

## Usage Instructions

### For New Projects

Simply run `/build-development-team` - skills will be created with proper frontmatter automatically.

### For Existing Projects

Run `/migrate-skills` to update existing skills:
```
/migrate-skills
```

The command will:
1. Scan all skills in `.claude/skills/`
2. Identify those needing frontmatter
3. Preview changes
4. Apply updates with backups

### To Modify Activation

Run `/toggle-skill-activation` to change how skills load:
```
/toggle-skill-activation [skill-name]
```
