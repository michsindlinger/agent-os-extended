# Anthropic Skills Format Migration - Lite Summary

> Created: 2026-01-12
> Full Spec: @agent-os/specs/2026-01-12-anthropic-skills-format-migration/spec.md

Migration der Agent OS Skill-Templates vom flachen Format (skill.md) zum offiziellen Anthropic Format (skill/SKILL.md) mit automatisiertem Migrations-Script fuer bestehende Dev-Teams.

## Key Points

- Templates migrieren von flachen Dateien zu Ordnerstruktur
- SKILL.md als zentrales Skill-Dokument (required)
- Optionale Unterordner: scripts/, references/, assets/
- Migrations-Script fuer bestehende Projekte
- Altes Format wird komplett ersetzt (keine Backward-Compatibility)
